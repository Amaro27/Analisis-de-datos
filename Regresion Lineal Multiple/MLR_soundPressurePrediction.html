Modelo de regresin lineal mltiple para la prediccin de presin de nivel escalado de presin sonora con datos de la NASA

Introduccin

En este trabajo entrenar y evaluar, con parmetros estadsticos pertinentes, un modelo de regresin lineal mltiple que busca predecir niveles de presin sonora detectada utilizando una base de datos originalmente publicada por la NASA [1]. Contiene 6 mediciones para cada una de las 1052 observaciones registradas en ella. Se encuentran:

  • Frecuencia (Hz)
  • ngulo de ataque (grados)
  • Longitud de cuerda geomtrica (m)
  • Velocidad de flujo libre (m/s)
  • Espesor del deplazamiento en el lado de succin (m)
  • Nivel escalado de presin sonora (dB)

De estos, se menciona que la nica salida es el nivel escalado de presin sonora; todos los dems son entradas en el experimento [1].

En concordancia con lo especificado, considerar como variable de inters (de salida) la presin sonora, y las dems variables como variables de entrada.

Teniendo lo anterior en cuenta, se calcular el $RSE$ y la $R^2$ del modelo para los datos de entrenamiento y los datos de prueba.

In[7]:
# datos de entrenamiento
rseTrain = results.scale**0.5
r2Train = results.rsquared

# datos de prueba
xtest = test.drop("presion", axis=1)
ytest = test.presion
yhatTest = results.predict(sm.add_constant(xtest))

mTest = xtest.shape[1]
nTest = xtest.shape[0]

rssTest = sum((ytest-yhatTest)**2)
rseTest = np.sqrt(rssTest/(nTest - mTest - 1))
tssTest = sum((ytest - np.mean(ytest))**2)
r2Test = 1 - rssTest/tssTest
print(f"RSE train: {rseTrain} RSE test: {rseTest}")
print(f"R^2 train: {r2Train} R^2 test: {r2Test}")

Esto es notable. Aun cuando podemos tener 95% de confianza (por la significancia de nuestro anlisis) de que hay una gran asociacin significativa entre las variables de entrada y la variable de salida, dado que $R^2$ es menor que $0.7$ (un criterio decente que usan diferentes investigadores), vemos que el modelo no es capaz de explicar adecuadamente el comportamiento de los datos obtenidos.

Conclusiones

Si bien los valores $p$ de las variables de entrada nos muestran que hay una asociacin muy significativa con la presin sonora detectada, tanto el coeficiente de $R^2$ obtenido como la Figura 1 muestran que el modelo no explica adecuadamente el comportamiento de los datos. Despus de todo, mientras ms se acerque la $R^2$ a $1$, ms deberan asemejarse los puntos de la grfica a una lnea recta como la de la Figura 1, una lnea de pendiente $1$

Esto qu nos ensea? Nos ensea que debe haber ms informacin sobre lo que impacte y contribuya al comportamiento de la presin sonora detectada. Muy probablemente haga falta considerar ms variables para acercarnos a mejores predicciones, o algn otro factor aleatorio que est impactando en la variable de salida.
