Introduccin

El objetivo de este trabajo es predecir la calificacin final de estudiantes de un curso a partir de su informacin demogrfica y de las calificaciones de los primeros 2 de 3 periodos de evaluacin, utilizando un modelo de MLR. Los datos se descargaron originalmente del Student Performance Data Set en el UCI Machine Learning Repository.

Estas son las categoras de la base de datos:

  • Escuela: Indica si el estudiante en cuestin asisti a la escuela Gabriel Pereira (GP) o a la escuela Mousinho da Silveira (MS).
  • Sexo: F para mujeres y H para hombres.
  • Edad: Edad del estudiante, en aos.
  • HorasDeEstudio: Cantidad de horas de estudio: 1 indica menos de dos horas, 2 indica de dos a cinco horas, 3 indica de cinco a diez horas, 4 indica ms de diez horas.
  • Reprobadas: Indica la cantidad de materias reprobadas previamente.
  • Internet: Si el estudiante tena acceso (yes) o no (no) a internet en su casa.
  • Faltas: Cantidad de veces que falt a clases.
  • G1: Calificacin del primer periodo, escala del 0 al 20.
  • G2: Calificacin del segundo periodo, escala del 0 al 20.
  • G3: Calificacin final, escala del 0 al 20.

Transformacin de datos categricos

Como se puede apreciar, los tipos de datos son los esperados. Por tanto, podemos empezar la transformacin de datos categricos (aun si son numricos) en datos booleanos a travs de one-hot encoding para que sean variables que nuestro modelo de MLR pueda manipular correctamente. En este caso, las categoras a transformar seran: Escuela, Sexo, HorasDeEstudio e Internet

In[163]:
# obtener indices de datos categricos
nombresDatosCategoricos = ["Escuela", "Sexo", "HorasDeEstudio", "Internet"]
indicesDatosCategoricos = df.columns.get_indexer(nombresDatosCategoricos)

# agregar 'l-1' variables dummy booleanas por cada variable categrica
for i in indicesDatosCategoricos:
    dummy = pd.get_dummies(df.iloc[:,i],drop_first=True)
    df = pd.concat([df,dummy],axis=1)

# quitar los datos categricos y mostrar nuevo df
df = df.drop(nombresDatosCategoricos,axis=1)
print(f"Vista preliminar: \n\n {df.head()}")

Valores atpicos

Ahora podemos explorar valores atpicos. En este caso explorar los valores de Faltas a travs del Mtodo de Tukey, para determinar si vale la pena examinarse en ms profundidad

In[164]:
# Variables de inters del mtodo
q1 = df.Faltas.quantile(0.25)
q3 = df.Faltas.quantile(0.75)
iqr = q3 - q1
limiteInferior = q1 - 1.5*iqr
limiteSuperior = q3 + 1.5*iqr

# Observaciones fuera de los lmites
outliers = df.loc[(df.Faltas < limiteInferior) | (df.Faltas > limiteSuperior), "Faltas"].values

De todos modos, cabe recalcar que el modelo tuvo una $R^2_{adj} > 0.86$, por lo que se puede decir que es adecuado para explicar en general la calificacin del tercer trimestre del ciclo escolar portugus en preparatorias dadas las variables de entrada consideradas.
