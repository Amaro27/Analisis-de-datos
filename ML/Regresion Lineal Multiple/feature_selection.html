Seleccin de caractersticas

Introduccin

Una de las reas dentro del aprendizaje automtico es la ingeniera de caractersticas (feature engineering), la cual es el "proceso de seleccionar, crear, o modificar caractersticas como variables de entrada o datos para ayudar a los modelos de aprendizaje automtico a aprender patrones de manera ms efectiva" [1]. Este trabajo se enfocar en la seleccin de caractersticas.

El objetivo de este trabajo es generar un modelo de regresin lineal mltiple que contenga solamente las variables seleccionadas por un proceso de seleccin hacia adelante y eliminacin hacia atrs.

Se trabajar con un conjunto de datos originalmente descargado de la UCI Machine Learning Repository [2], y la descripcin de las variables es la siguiente:

  • acidezFija. La acidez fija del vino, medida en gramos de cido tartrico por decmetro cbico.
  • acidezVolatil. La acidez voltil del vino, medida en gramos de cido actico por decmetro cbico.
  • acidoCitrico. Gramos de tcito ctrico por decmetro cbico.
  • azucarResidual. Gramos de azcar por decmetro cbico.
  • cloruros. Gramos de cloruro de sodio por decmetro cbico.
  • dioxidoAzufreLibre. Miligramos de dixido de azufre libre por decmetro cbico.
  • dioxidoAzufreTotal. Miligramos de dixido de azufre total por decmetro cbico.
  • densidad. Medida en gramos por centmetro cbico.
  • pH. Valor del vino en la escala de pH.
  • sulfatos. Gramos de sulfato de potasio por decmetro cbico.
  • alcohol. Volmen percentil de alcohol en el vino.
  • calidad. Mediana de la calidad otorgada por al menos tres catadores, en escala del 0 (muy malo) al 10 (excelente).

En este caso, se buscar entrenar un modelo que pueda predecir la calidad de vino de acuerdo con las dems variables explicativas.

El mismo sitio web no es menciona que no hay valores faltantes en el conjunto de datos.

Modelo base

Para poder apreciar el efecto de la seleccin de caractersticas, entrenemos un modelo base y evaluemos su desempeo:

In[5]:
linearModel = LinearRegression()
linearModel.fit(xTrain, yTrain)
yHatTestBase = linearModel.predict(xTest)
print(f"R^2 del modelo base: {r2_score(yTest, yHatTestBase)}")

R^2 del modelo base: 0.3283887639581424

Debido a que los mtodos de seleccin de caractersticas reducirn los modelos, la $R^2$ no ser una mtrica adecuada para comparar los modelos, ya que esta tiende a aumentar al agregar ms variables. Por lo tanto, usaremos la $R^2$ ajustada para ello. Veamos la $R^2$ ajustada del modelo base:

In[6]:
def r2_adj(r_2, n, m):
    # n: nmero de observaciones
    # m: nmero de variables explicativas
    r_2_adj = 1 - (1 - r_2) * (m - 1) / (m - n - 1)
    return r_2_adj

r2adj_base = r2_adj(r2_score(yTest, yHatTestBase), N, M)
print(f"R^2 ajustada del modelo base: {r2adj_base}")

R^2 ajustada del modelo base: 0.3233072161444588

Forward Selection

Se utiliza el proceso de forward selection para reducir las caracteristicas del modelo, y se usa validacin cruzada para que pueda generalizar mejor

Conclusiones

Se pudo ver cmo la seleccin de caractersticas pudo ayudar a reducir las variables que el modelo considera con diferencias mnimas en desempeo. En este caso se redujo un modelo con 11 variables explicativas a uno con 4. Si bien puede no parecer mucho, esto puede significar entre tener un modelo de 10,000 variables con una $R^2$ ajustada de 93% a uno de 500 variables con una $R^2$ ajustada de 91%, lo cual puede ser una gran diferencia en cuanto a costos computacional y tiempos de entrenamiento. Dependiendo de la aplicacin en la que se quiera implementar un modelo se determinar cunto ajuste se est dispuesto a sacrificar para reducir el tamao del modelo, pero al menos este trabajo pudo demostrar el impacto que puede tener la seleccin de caractersticas para reducir la dimensionalidad de un modelo a cambio de una mnima prdida en desempeo.
